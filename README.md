# 华润三九 AI用药伴侣 Demo

专为 55-75 岁中老年人设计的智能用药辅助系统。本项目通过集成 **火山引擎豆包大模型 (Doubao)**，实现了基于图像识别（OCR）的自动药品录入、智能服药排程、以及通俗易懂的用药知识问答功能。

## 核心功能 (Features)

1. **AI 视觉识药 (Vision OCR)**
   - 老年人只需拍摄药盒或说明书，AI 自动提取药品名称、单次剂量及建议服药时间。
   - 提取结果自动存入“电子药箱”，免去繁琐的手动打字输入。
2. **智能排程与打卡 (Smart Scheduling)**
   - 根据 AI 识别的建议服药频率，自动在首页生成当天的服药计划表。
   - 服药后“一键打卡”，系统自动同步扣减药箱内的药品库存，当库存不足时给予红色警示。
3. **用药答疑与预警 (AI Consultation)**
   - 随时通过语音或文字向豆包 AI 提问（如：“阿托伐他汀能空腹吃吗？”）。
   - AI 被配置为“关怀模式”，用词极简、通俗，并且在解答时**强制要求提示常见副作用**及安全注意事项。

## 技术栈 (Tech Stack)

- **前端**：原生 HTML5 + CSS3 + Vanilla JavaScript (无需打包，开箱即用)
- **后端代理**：Node.js + Express.js (主要用于解决浏览器跨域 CORS 问题，并安全封装大模型 API 密钥)
- **AI 大模型**：火山引擎豆包大模型 (模型 ID: `doubao-seed-2-0-mini-260428`)

## 快速启动 (Getting Started)

### 1. 环境准备
请确保您的电脑上已安装 [Node.js](https://nodejs.org/) (建议版本 v16+)。

### 2. 下载依赖
在项目根目录运行以下命令安装必需的依赖（如 `express`, `cors`, `axios`）：
```bash
npm install
```

### 3. 启动本地代理服务
启动后端代理服务器，这会将您的前端请求安全地转发给火山引擎 API：
```bash
npm start
```
*控制台将输出服务已启动的信息，默认监听 `3000` 端口。*

### 4. 访问体验页面
代理服务启动后，直接在浏览器中打开以下地址即可体验：
```text
http://localhost:3000/demo.html
```

## 设计理念

为了最大化适应中老年人的生理和认知特点，本产品UI严格遵循：
- **极简操作**：去除了所有深层嵌套菜单，采用直观的底栏导航。
- **高容错**：输入出错时提供温和的重试提示，数据支持随时点击快捷修改。
- **强反馈**：操作成功后有明显的颜色反馈及弹窗提示，避免老年人产生“不知道有没有点上”的疑虑。
